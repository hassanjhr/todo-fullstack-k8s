# Frontend Development Skill

## Instructions

1. **Page & Layout Structure**
   - Build semantic page structures
   - Create reusable layouts
   - Use responsive grid or flex systems
   - Maintain consistent spacing and alignment

2. **Component Development**
   - Design reusable UI components
   - Pass props and manage component state
   - Keep components small and focused
   - Follow naming conventions

3. **Styling & Responsiveness**
   - Apply consistent design tokens
   - Use CSS modules or scoped styles
   - Ensure mobile-first responsiveness
   - Handle hover, focus, and active states

4. **User Experience**
   - Optimize for accessibility
   - Provide visual feedback on interactions
   - Ensure fast load and render times
   - Handle empty and loading states

## Best Practices
- Build reusable and composable components
- Keep styles modular and maintainable
- Avoid layout shifts
- Test UI on multiple screen sizes
- Follow accessibility standards (ARIA, contrast)

## Example Structure

```html
<section class="page">
  <header class="navbar">Navbar</header>

  <main class="content">
    <h1 class="title">Page Title</h1>
    <div class="card">
      <p>Reusable component</p>
      <button class="primary-btn">Action</button>
    </div>
  </main>
</section>
